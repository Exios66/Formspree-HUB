Frequently Asked Questions

How do I upload a file?

Simply drag and drop your file into the upload area or click to browse your files. Select the file you want to upload and click the "Upload" button.

What file types are supported?

We support various file types including images (JPG, PNG, GIF), documents (PDF, DOC, TXT), and archives (ZIP, RAR).

What is the maximum file size?

The maximum file size limit is 5MB per file.

How long are files stored?

Files are stored for 30 days from the upload date.
